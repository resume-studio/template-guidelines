## Document

Requirements:
- Size: 595pts x 842pts
- Font: any from [Google Fonts](https://fonts.google.com/)
- Minimum font size: 11pts
- Grid: 4pts

## Sections

All fields/sections are optional, so a missing field/section should not break the design.
